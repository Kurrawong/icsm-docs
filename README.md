# ICSM Documents

This repository contains early work to realise [Intergovernmental Committee on Surveying & Mapping (ICSM)](https://wwww.icsm.gov.au) documents, such as the _Australian Place Naming Principles_ and the _Addressin 2035_ strategy as Linked Data resources for potential future publication via the ICSM's Linked Data platform tool at <https://icsm.information.qld.gov.au/>.

This work is being done by [KurrawongAI](https://kurrawong.ai) based on a requirement to better deliver the 

## Contents

The plan is to deliver ICSM documents as Linked Data resources with their constituent parts delivered as other Linked Data resources, principally vocabularies. For example the _Australian Place Naming Principles_ document can be represented as a `schema:DigitalDocument` which contains two vocabularies: a glossary of Place Name terms and one of _Principles_ for place naming.
